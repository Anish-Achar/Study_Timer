# Study_Timer
The study timer is a modern, interactive web-based productivity tool designed to help students track and optimise their study sessions with both functionality and visual engagement. It combines a clean, minimal interface with advanced features such as subject tracking, session logging, and performance analytics, making it more than just a basic timer.

At its core, the application offers two primary modes: a stopwatch and a countdown timer. The stopwatch allows users to measure open-ended study sessions, while the timer enables structured sessions, similar to techniques like the Pomodoro method. The interface is centred around a large, highly visible time display, ensuring that users can easily track progress without distraction.

One of the most distinctive features is the subject selection system. Instead of a traditional dropdown, subjects are accessed through a gradient circular button. When hovered over, this expands into a semi-circular layout of subjects, creating a smooth and visually appealing interaction. Selecting a subject dynamically changes the colour theme of the application, reinforcing focus and providing a sense of context for each session.

The timer also includes intelligent session logging. As users switch subjects or complete sessions, time is automatically recorded and stored locally. This data feeds into a dedicated statistics dashboard, where users can analyse their study habits. The dashboard provides insights such as total study time, subject breakdown, weekly activity, and time-of-day patterns. Additionally, users can manually add sessions, delete entries, or export their data to Excel for further analysis.

Visually, the application stands out through its animated background, featuring aurora-style gradients, subtle particle effects, and smooth transitions. These elements create an immersive yet non-distracting environment that enhances the overall user experience.

Overall, this study timer blends productivity with thoughtful design. It not only helps users manage their time effectively but also provides meaningful insights into their learning patterns, making it a powerful tool for consistent and focused study.
